# MIoT-Dashboards


## Steps to deploy dashboards:
    * Activate miot_dash_venv (or simply create one):
        * >miot_dash_venv/Scripts/activate
    
    * Install dependencies:
        * >pip install -r requirements.txt
    
    * Run app:
        * >python app.py
        * navigate to http://127.0.0.1:8888/
